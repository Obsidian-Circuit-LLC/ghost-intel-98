# Ghost Intel 98 v3.26.0 — Windows smoke checklist

A quick click-through to confirm the build is solid on your machine. Do them in order; each line is one action. If anything doesn't behave as described, note the step number and tell Desirae — that's exactly what this catches.

**What's automatically verified already:** every tool is registered and reachable (no hidden/orphaned tools), and all four new features are confirmed present in the installed build. This checklist covers the *runtime* half — that each tool actually opens and works on real Windows.

## 0. Install & boot
- [ ] Install `GhostIntel98-Setup-3.26.0.exe` (More info → Run anyway; it's unsigned).
- [ ] App launches to the desktop with no error dialog.
- [ ] Nothing flashes an "ERR_REQUIRE_ESM" / white-screen / crash on startup.

## 1. AI Assistant — free text selection (new)
- [ ] Open **AI Assistant**. Send any short message and let it reply.
- [ ] **Click-drag to highlight** part of a reply, press **Ctrl+C** — it copies (this is the fix).
- [ ] **Right-click** a message — the **Copy message / Copy whole conversation** menu still appears.

## 2. Adaptive Memory (new — off by default)
- [ ] Settings → find **Case Memory / Adaptive memory** toggles. They're **off by default** — that's intended.
- [ ] Turn on **Adaptive memory** (needs the local Ollama AI). Have a short chat.
- [ ] Open the assistant's **Memory panel** — you can see what was recalled and any "learned" items.
- [ ] Edit/pin/delete a learned item, and **Wipe** — confirm items actually disappear (nothing lingers).

## 3. OSINT Toolkit (new)
- [ ] Double-click the **OSINT Toolkit** desktop icon (also in the Access menu).
- [ ] It opens a folder-style window grouping tools: **Social Media / Geospatial / Identity / Network-Recon**.
- [ ] Every tool you expect is listed (X, GhostScrape, SOCMINT, GeoINT, EyeSpy, Camera, Searchlight, Host Info, Net Explorer, News). Nothing missing.
- [ ] Click a tile — it opens that tool's window.

## 4. GhostScrape (new — X timeline scraper)
- [ ] Open **GhostScrape** (from the toolkit or its own window).
- [ ] Before enabling the X network gate, the Start button shows a **visible reason** it's blocked (not a dead button).
- [ ] Pick a stored X account (shared with X Intel), enter a username, choose a type, and run a small scrape.
- [ ] Results appear in the table; **Export** (JSON/TXT/CSV) works; **Save to case** attaches to a case.
- [ ] If it fails, the message is a plain sentence (no file paths / tokens leaked on screen).

## 5. Searchlight — sweep persistence (fixed)
- [ ] Open **Searchlight**, run a username sweep, let results start coming in.
- [ ] Switch to another Searchlight tab (Graph/Reports) and **back to Sweep** — your results are **still there** (this was the bug).
- [ ] Switch to a **different case** and back — each case shows its own sweep, not a blank panel.

## 6. The rest of the tools open cleanly
Open each and confirm it renders with no error:
- [ ] My Cases · Notepad 98 · Briefcase · Bookmarks · Calendar · Reminders
- [ ] Chat (beta) · Mail · DialTerm
- [ ] GeoINT (3D map loads) · EyeSpy (camera wall) · Camera · News · Host Info · Net Explorer
- [ ] SOCMINT · X / Twitter
- [ ] Games ▸ (Solitaire, Mine Detector, Chess, Ghost Space Ball)

## 7. Report
- [ ] Anything that errored, looked broken, or was missing → note the **step number** and a one-line description.

*Everything from v3.25.0 carries forward. This build was gated on a machine-verified reachability audit + a packaged-artifact integrity check; this checklist is the human runtime pass. A fully-automated Windows VM smoke is planned for the next cycle.*
